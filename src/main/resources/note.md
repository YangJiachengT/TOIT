mysql left join 逗号分隔字段查询

goods表
+----------+-------------+
|    id    |  school_ids |
+----------+-------------+
| 	  1    |    1,2,3    |
| 	  2    |      2      |
|     3    |      2      |
+----------+-------------+
school表
+----------+------------------+
|    id    |  locality_name   |
+----------+------------------+
| 	  1    |    西安科技大学   |
| 	  2    |    西安欧亚学院   |
|     3    |    西安邮电大学   |
+----------+------------------+

SQL：

select g.id,g.school_ids,GROUP_CONCAT(s.locality_name  ORDER BY s.id ASC) 
from goods g 
	left join school s on FIND_IN_SET(s.id , g.school_ids)
GROUP BY g.id
ORDER BY g.id ASC

结果：
+----------+---------------+------------------------------------+
|    id    |  school_ids   |   GROUP_CONCAT(s.locality_name)    |
+----------+---------------+------------------------------------+

| 	  2    |      2        |西安欧亚学院                         |
|     3    |      2        |西安欧亚学院                         |
